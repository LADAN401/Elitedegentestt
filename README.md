# Smart Wallet Intent Decoder - Demo Bot

This is a demo Telegram bot that simulates **DEX swap events** and sends **Smart Wallet Intent alerts**.  

## Features

- Demo swaps for multiple wallets
- Intent decoding:
  - Sniper Entry
  - Accumulation
  - Stealth Dump
  - Profit Taking
- Formatted Telegram alerts

## Setup

1. Clone the repository
2. Run `npm install`
3. Copy `.env.example` to `.env` and add your Bot Token and chat/channel
4. Run the bot: `npm start`
